How to export your server list

  1. Open Discord in your browser and log in
  2. Open Developer Tools (F12 or Cmd+Option+I)
  3. Go to the Console tab
  4. Paste and run this code:
fetch('/api/v9/users/@me/guilds')
  .then(r => r.json())
  .then(d => {
    const a = document.createElement('a');
    a.href = URL.createObjectURL(
      new Blob([JSON.stringify(d, null, 2)])
    );
    a.download = 'guilds_api.json';
    a.click();
  });

This will download guilds_api.json with your server list.
